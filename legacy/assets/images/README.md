# Images Directory

Placez vos images personnalisées dans ce dossier.

## Images nécessaires pour la section "Pourquoi Nous"

1. **pourquoi-image-1.jpg** - Image en haut (consultation d'affaires/avocat)
   - Dimensions recommandées: 600x400px
   - Format: JPG, PNG ou WebP

2. **pourquoi-image-2.jpg** - Image en bas (consultation légale)
   - Dimensions recommandées: 600x350px
   - Format: JPG, PNG ou WebP

## Utilisation

Remplacez les URLs Unsplash dans `index.php` par vos propres images:

```php
<img src="assets/images/pourquoi-image-1.jpg" alt="Business consultation">
<img src="assets/images/pourquoi-image-2.jpg" alt="Legal consultation">
```
